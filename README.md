# THE SANJALIKA WATERPARK: <br> - First semester project


## <u>ABOUT</u>
The Sanjalika waterpark is definitely an ultimate and out of this world source of entertainment decorated by mermaids, statues and has the largest cave and swimming area of modern technology. A kind of place for thrills and fun! <br> At the Sanjalika waterpark there is lots of water rides which are very exciting and thrilling.

<br>


*SPECIFICATIONS:*
- A mobile responsive website which can works efficiently on various screen sizes.

- Navigation throughout the website is smooth.

- A timing feature of different slides available at the website.

- A well animated representation of all slides and rides at the waterpark.

- The website includes a booking section in order to book slides in the waterpark.

- The website includes a GeoLoaction API of the address of the waterpark in the "Contact Us" section of the website.
